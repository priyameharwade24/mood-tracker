# Mood Tracker

## Project Overview
Mood Tracker is a simple and interactive web application that helps users record and monitor their daily moods. The application provides an easy way to track emotions and maintain personal well-being records.

The project is developed using React and Vite for fast performance and responsive user experience.

---

## Features
- Track daily moods
- Interactive and user-friendly interface
- Real-time mood updates
- Responsive design
- Simple navigation

---

## Technologies Used
- React
- Vite
- JavaScript
- HTML
- CSS

---

## Project Structure
- `src/` → Main React components
- `public/` → Static assets
- `App.jsx` → Main application component
- `main.jsx` → React entry point

---

## How to Run the Project

1. Install dependencies

```bash
npm install
```

2. Start development server

```bash
npm run dev
```

3. Open the localhost link in your browser
